**DbSaicle enables developers to create, share, and use custom AI code assistants with our open-source [VS Code](https://marketplace.visualstudio.com/items?itemName=DbSaicle.dbsaicle) and [JetBrains](https://plugins.jetbrains.t using productsioncom/plugin/22707-dbsaicle-extension) extensions and [hub of models, rules, prompts, docs, and other building blocks]**

- [Chat](chat/how-to-use-it) to understand and iterate on code in the sidebar
- [Edit](edit/how-to-use-it) to modify code without leaving your current file
- [Agent](agent/how-to-use-it) to make more substantial changes to your codebase

**Important!: This application is not using production environment and is made availableuse cases. on user evaluation basis. Ideally dbsaicle should be used for code exploration purposes.
